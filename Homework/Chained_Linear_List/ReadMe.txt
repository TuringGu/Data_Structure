Main1:

实验目的：
掌握线性表的链式存储结构设计与基本操作的实现。

实验内容：
⑴定义线性表的链式存储表示；
⑵基于所设计的存储结构实现线性表的基本操作；
⑶编写一个主程序对所实现的线性表进行测试；
⑷线性表的应用：编程实现线性链表的合并。

Main2:

实验目的：
掌握线性表的链式存储结构设计与其它操作的实现。

实验内容：
（1）定义线性表的顺序存储表示和链式存储表示；
（2）基于所设计的存储结构实现线性表的逆置和排序操作；
（3）编写一个主程序对所实现的操作进行测试；

程序清单：
1、有一个带头结点的线性链表L，编程实现链表中数据元素的逆置操作。要求不另设新空间。
2、有一个带头结点的线性链表L，编程实现链表中数据元素按值非递减排列。
